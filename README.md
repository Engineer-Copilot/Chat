# Chatbot

**1. Install Dependencies**

```bash
yarn install
```

**2. Provide Keys**

Find the .env file in the root of the repo. Add the following variables

```bash
OPENAI_API_KEY=''
PINECONE_API_KEY=''
PINECONE_ENVIRONMENT=''

# Navigate to Indexes under your Project to retrieve the Index name
PINECONE_INDEX=''
```

**3. Run App**

```bash
yarn dev
```

**5. Use chatbot**

Begin chatting.
